GitHub заблокировал загрузку, обнаружив реальные API токены
Shippo в коде. Нужно их удалить или заменить на примеры.

═══════════════════════════════════════════════════════════════
  РЕШЕНИЕ 1: РАЗБЛОКИРОВАТЬ ЧЕРЕЗ ССЫЛКУ
═══════════════════════════════════════════════════════════════

GitHub предоставил ссылку для разблокировки:

https://github.com/bryanwax12/newbotcursor/security/secret-scanning/unblock-secret/35uywZn5DgxSby9knIYoWry2BEO

1. Откройте эту ссылку в браузере
2. Подтвердите, что хотите разрешить этот секрет
3. После этого выполните снова: git push -u origin main --force

═══════════════════════════════════════════════════════════════
  РЕШЕНИЕ 2: УДАЛИТЬ СЕКРЕТЫ ИЗ КОДА
═══════════════════════════════════════════════════════════════

Нужно найти и заменить все реальные токены на примеры:

1. Найдите файлы с токенами Shippo
2. Замените реальные токены на "shippo_test_xxx" или "your_shippo_key"
3. Убедитесь, что .env файл в .gitignore
4. Создайте новый коммит
5. Выполните push

═══════════════════════════════════════════════════════════════
  РЕШЕНИЕ 3: ЗАГРУЗИТЬ БЕЗ ИСТОРИИ
═══════════════════════════════════════════════════════════════

Создать новый репозиторий без истории:

1. Удалите папку .git
2. Создайте новый git репозиторий
3. Убедитесь, что все секреты удалены
4. Загрузите заново

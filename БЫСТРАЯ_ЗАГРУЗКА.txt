  БЫСТРАЯ ЗАГРУЗКА НА GITHUB
═══════════════════════════════════════════════════════════════

СПОСОБ 1: ПРОСТОЙ (РЕКОМЕНДУЕТСЯ)
═══════════════════════════════════════════════════════════════

1. Дважды кликните на файл: ЗАПУСТИТЬ_УСТАНОВКУ.bat
2. Следуйте инструкциям на экране

═══════════════════════════════════════════════════════════════
СПОСОБ 2: ЧЕРЕЗ POWERSHELL
═══════════════════════════════════════════════════════════════

1. Откройте PowerShell в этой папке
2. Выполните команду:

   powershell -ExecutionPolicy Bypass -File install_git_and_upload.ps1

Или:

   .\install_git_and_upload.ps1

═══════════════════════════════════════════════════════════════
СПОСОБ 3: РУЧНОЙ (ЕСЛИ СКРИПТЫ НЕ РАБОТАЮТ)
═══════════════════════════════════════════════════════════════

1. Установите Git: https://git-scm.com/download/win
2. Откройте PowerShell в этой папке
3. Выполните:

   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/bryanwax12/newbotcursor.git
   git branch -M main
   git push -u origin main

═══════════════════════════════════════════════════════════════
СПОСОБ 4: ЧЕРЕЗ ВЕБ-ИНТЕРФЕЙС GITHUB
═══════════════════════════════════════════════════════════════

1. Откройте: https://github.com/bryanwax12/newbotcursor
2. Нажмите "Add file" → "Upload files"
3. Перетащите ВСЕ файлы из этой папки (кроме .env)
4. Нажмите "Commit changes"

  ЗАГРУЗКА ЧЕРЕЗ ZIP-АРХИВ
═══════════════════════════════════════════════════════════════

GitHub ограничивает загрузку до 100 файлов за раз.
Решение: загрузить ZIP-архив и распаковать его на GitHub.

═══════════════════════════════════════════════════════════════
  СПОСОБ 1: ЗАГРУЗИТЬ ZIP-АРХИВ
═══════════════════════════════════════════════════════════════

1. Я создал архив: C:\Users\super\telegram-shipping-bot.zip

2. Откройте: https://github.com/bryanwax12/newbotcursor

3. Нажмите "Add file" → "Upload files"

4. Перетащите файл: telegram-shipping-bot.zip

5. В поле "Commit changes" введите:
   Initial commit: Modern Telegram shipping bot

6. Нажмите "Commit changes"

7. После загрузки GitHub автоматически распакует архив!

═══════════════════════════════════════════════════════════════
  СПОСОБ 2: УСТАНОВИТЬ GIT (РЕКОМЕНДУЕТСЯ)
═══════════════════════════════════════════════════════════════

С Git можно загрузить все файлы сразу без ограничений:

1. Установите Git: https://git-scm.com/download/win
2. После установки запустите: ЗАПУСТИТЬ_УСТАНОВКУ.bat
3. Или выполните команды в PowerShell:

   cd C:\Users\super\telegram-shipping-bot
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/bryanwax12/newbotcursor.git
   git branch -M main
   git push -u origin main

═══════════════════════════════════════════════════════════════
  СПОСОБ 3: РАЗБИТЬ НА ЧАСТИ
═══════════════════════════════════════════════════════════════

Если архив не работает, можно загрузить по частям:

1. Сначала загрузите основные файлы:
   - README.md
   - requirements.txt
   - pyproject.toml
   - .gitignore
   - .env.example

2. Затем загрузите папку src/ (по частям, если много файлов)

3. Затем остальные файлы
